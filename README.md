## lecture 11 : 호, 타원, 한계

1. 호를 그리는 5가지 방법 : 호는 기본적으로 반시계 방향으로 그려진다.

   1.1. 호 아이콘 클릭 후 세 개의 점을 선택 : **A + space**, 여기서 직교모드가 켜져있으면, 호가 그려지지 않는다.

   ​	   A space : 호를 그리는 모드에 진입

   ​	   E space : 끝점 지정 모드

   ​	   끝점 지정 후, 각도 (A), 반지름 (R), 방향(D)를 활용하여 그릴 수 있다.

   1.2. 중심점 활용 방법

   ​	  A space + C space 후에는, 호의 중심점 선택 --> 호의 시작점 선택 --> 각도 값 입력

   

2. 타원 그리는 방법에는 세 가지가 존재한다.

   2.1. 단축키 : **EL + space**를 하게 되면, 먼저 타원의 시작점 및 끝점을 선택하고, 임의로 길이를 선택

   2.2. 같은 과정을 거치고, 중심점으로부터의 길이값을 입력하여, 그리는 방법

   2.3. EL + space + C + space : 타원의 중심점을 설정하는 방법 --> 후에, 끝점에 대한 거리값을 입력하고, 중심으로 부터의 거리값을 입력하면 된다.

   

3. 한계 설정 (Limit)

   3.1. 명령창에 LIMITS 입력 : 왼쪽 아래 구석 지정을 하면 된다. 또는 명령창에 바로 좌표값 e.g. 0,0을 입력하면 된다.

   3.2. 그 다음, 오른쪽 위 구석을 지정해야 한다. 이때도 클릭해도 되고, 좌표를 직접 입력해도 된다.

   3.3. 한계 영역을 설정한 다음, LIMITS - ON을 입력하게 되면, 벗어난 경우 ** 외부 한계라는 것이 뜨게 된다.

   3.4. 해당 기능은, 출력 영역을 설정할 때, 주로 활용하기도 한다.

   ​		**ctrl + P**를 하게 되면, 출력창이 나오고, 플롯 대상을 한계로 설정 및 플롯의 중심 조건을 설정하면 다음과 같이 나온다.

## lecture 19 : 문자 및 치수 기입

1. 문자를 기입하는 Text 기능의 단축키 : **T + space** / **DT + space** (문자 규격 수정가능 / 한줄 문자) 

   * 작업을 완료할 때에는, ctrl + enter를 눌러야 한다.
   * 문자의 글꼴이나 스타일을 지정하고자 할 때에는, DT + space + 자리 맞춤 옵션 or 여러 세부적인 기능들을 활용하면 된다.

   * 문자를 편집할 때에는 텍스트를 더블 클릭하면 되고, 규격을 설정한 뒤에 T + space를 하게 되면, 적용이 되어있다.
   * **문자의 그립점 형태 변경 방법**
     * 문자의 열을 '열없음'으로 설정하면, 그립의 형태가 변경된다.
     * 문자의 그립 형태를 변경함으로써, 문자를 더욱 효율적으로 작성할 수 있다.



2. 치수 기입

   2.1. 선형 치수 : **DLI + space**  /  기울어진 선의 경우 : **DAL + space**

   space를 한번 더 입력하게 되면, 객체를 선택하는 옵션으로 넘어가게 된다.

   2.2. 반지름 치수 : **DRA + space**

   ＃치수가 나오는 숫자의 텍스트 크기를 조절하고 싶다면, **DDIM + space**를 입력

   2.3. 지름 치수 : **DDI + space**

   2.4. 각도 치수 : **DAN + space**

   2.5. 신속 치수 : **QD + space** 후, 객체를 선택하고 원하는 지점을 클릭하게 되면, 치수가 측정이 된다.

   2.6. 기준선 치수 : **DBA + space **후, 기준 치수선을 선택하고, 원하는 지점을 클릭하게 되면, 기준선으로 부터의 거리가 나온다.

   2.7. 연속 치수  : **DCO + space** 후, 선형 치수를 먼저재고, 끝지점을 기준점으로 다음 치수를 기입

   2.8. 신속 지시선 : **LE + space** 후, 점을 지정하고 space를 누르는 것이 아니라 enter를 눌러야 다음 옵션으로 넘어간다.

   자주 사용하는 치수 : **%%c** (지름 입력을 위함)

### lecture 30 : 간격 띄우기, 자르기, 회전, 축척 (offset, trim, rotate, scale)

1. offset 단축키 : **O + space** 

   그 다음, offset을 어느 정도 줄지에 대한 수치 기입 및 객체를 선택하게 된다. 그렇게 되면, offset이 들어간 만큼 같은 객체가 생성되게 된다.

   선, 사각형 모두 적용가능하다. 

2. 자르기 단축키 : **TR + space**

   자르기를 진행하게 되면, 객체를 하나씩 선택해도 되고, drag를 진행하게 되면, 걸리는 객체마다 모두 선택 지정이 된다.

   TR + space + **T + space**를 입력하고, 기준 객체를 선택한 후 space를 하게 되면, 기준 객체를 기준으로 trim을 진행하게 된다.

3. 객체 회전 단축키 : **RO + space**

   RO + space를 친 후, 회전할 객체 및 기준점 또는 섬을 순차적으로 선택한다. 그 다음, 원하는 각도를 입력하면 된다. 이때, 객체의 회전 기능이기 때문에, 원본이 복사되는 것은 아니다.

   ＃ 마지막에, 복사를 기능을 넣어주게 되면, 원본 객체에 대한 복사도 진행된다. 

   ＃ 원본에 대한 객체에 대해서 복사를 하는 순서 : **RO + space / 객체 선택 + space / 기준점 선택 + space / <u>C +space</u> / 각도 입력**

4. 축척 단축키 : **SC + space**

   후에, 객체 클릭 후, 기준점 및 scale factor를 지정해주면 된다.

### lecture 38 : 연장, 정보, 분해, 모따기, 모깎기(Extend, List, Explode, Chamfer, Fillet)

1. 객체 연장 단축키 : **EX + space**

   1.1. EX + space + **B + space (경계 모서리)**

   * 해당 명령어 후, 경계 모서리를 지정한 후, 연장하고자 하는 객체를 선택하게 되면, 기준이 되는 객체에 대해 연장이 되게 된다.



2. 객체에 대한 정보를 얻을 수 있는 단축키 : **LI + space**

   명령어 입력 후, 객체를 선택하고 space를 누르게 되면, 면적, 폭, 둘레 및 각 점의 절대 좌표 등등 객체에 대한 상세한 정보가 나오게 된다.



3. 분해 단축키 : **X + space**

   예시) X + space 후, 사각형 객체를 적용하게 되면, 사각형 객체는 직선 객체 총 4개로 분해되게 된다.



4. 모따기 (Chamfer 기능) 단축키 : **CHA + space**

   4.1. CHA + space + **D + space (거리 옵션)** + 각 거리 (a, b) 입력 후, 객체를 순차적으로 선택하게 되면, 두 객체가 교차하는 지점으로부터 첫 번째 객체까지 a만큼, 교차 지점으로부터 두 번째 객체까지 b만큼 모따기가 실행된다.

   <img src="https://github.com/HanMinung/DLIP/assets/99113269/58c618bc-a798-4e86-9dee-e68b3b2a63d4" alt="image" style="zoom:50%;" />



5. 모깎기 (Fillet) 단축키 : **F + space**

   여기에도, 여러 옵션들이 존재하기 때문에, 존재하는 옵션들을 활용하여 fillet을 수행하면 된다. 



### lecture 44 : lecture 44 : 측정, 증분, 끊기, 줌, 신축(Dist, Lengthen, Break, Zoom, Stretch)

1. 거리 측정 명령어 : **DI + space**

   

2. 객체의 거리를 늘리거나 줄이는 명령어 : **LEN + space**

   옵션에서, 증분 / 퍼센트 등등의 옵션들을 선택해서 길이를 늘리거나 줄이는 작업을 수행할 수 있다.

   단순하게 + -를 입력하여 증분 옵션을 주면 된다.

   

3. 객체를 끊는 break 명령어 : **BR + space**

   BR + space 후, 객체를 끊을 시작점 및 끝점을 클릭하면 된다.

   여기서 끊어지는 선이 생성되는 것은 반시계 방향 기준이다.

   

4. 줌기능 : **Z + space**

   default : 첫 번째점 클릭 / 두 번째점 클릭을 하게되면 해당 영역을 확대하게 된다.

   

5. 객체를 늘리거나 줄이는 신축 기능 (객체의 형태를 수정할 때 사용하기 때문에 중요) : **S + space**

   실행 후, 객체가 선택될 수 있도록 영역을 지정해준 다음, 기준점을 선택한 다음, 방향 지정 및 정확한 거리를 지정해주면 된다.

   LEN 기능과 다른 점은, + -를 따로 구분하지 않고, 사용자에 맞게 길이를 수정할 수 있다는 장점이 있다. 

   

### lecture 49 : 배열, 해치, 도면층 특성 관리자 (Array, Arrayclassic, Hatch, Layer)

1. 배열 : Fusion 360에서의 pattern과 같은 옵션

   ARRAYCLASSIC 기능 (단축키 X) / ARRAY (단축키 : **AR + space**) 두 가지 기능으로 사용하는 방법이 있다.

   

   1.1. ARRAYCLASSIC 기능

   실행하면, 창이 뜨는데, 창에서 배열의 형태 지정 / 행 및 열의 크기 / 행간의 거리, 열간의 거리를 설정하여 활용할 수 있다.

   회전 시키면서 복사를 진행 On/Off에 따라 생성되는 배열의 형태가 달라지니 잘 선택해야 한다.

   

   1.2. **AR + space**

   여기서는, 위의 창에서 선택했던 모든 옵션들이 단축키로 들어가 있는 최신 버전

   항목 수 / 사이 각도 등에 대한 옵션들이 명령창에 뜨니, 그런것들을 보고 원하는 형상을 제작하면 된다.

   그리고, 중심점을 클릭하게 되면, 위의 옵션바에 설정하는 배열 관련 설정창이 뜨게 된다.

   

2. 해치 기능 : **H + space**

   기계제도에서는, 절단면을 표시할 때, 해치 기능을 활용하여 표시한다.

   이미 설정된 해치면의 표시 스타일을 수정하고자 한다면, 해치 수정 **HE + space**을 하면 된다.

   최신 버전에서는, H + space 후, 객체를 지정하게 되면, 상단바에 설정 옵션들이 나오는데, 설정 후 닫기만 하면 된다.



### lecture 54 : 블록, 삽입, 선축척, 치수축척, 특성, 특성복사(Block, Insert, Ltscale, Dimscale, Properties, Matchprop)

1. 블록 기능 단축키 : **B + space**

* 자주 사용하는 기호/ 도면 양식 등을 블록으로 저장하여, 필요할때마다 손쉽게 불러올 수 있도록 하는 기능

* 블록으로 지정하게 되면, 두 객체는 하나의 객체로 인식이 되게 된다.

* 블록 편집기능 : **BE + space**
* 블록을 소거하는 기능 : **PU + space**에서 삭제



2. 삽입 단축키 : **I + space**

* 주로 만들어진 블록을 삽입할 때 사용하는 기능



3. 선축척 단축키 : **LTS + space**

* 도면층 특성 관리자 단축키에서 설정을 해두더라도, 실제 객체의 크기에 따라 선의 축척을 조절할 필요가 있는 경우가 있다.
* 그러한 경우에, 선축척을 조절하기 위해, 주로 이 기능을 활용
* 1보다 작은 숫자를 입력하면, 기존의 스케일보다 더 작아진다.



4. 치수 축척 단축키 : **DIMSCALE + space**

* dimscale + space 후, 새로운 스케일을 입력하게 되면, 후에 나오는 치수 정보에 대한 표기들에 반영이 된다.

* 이때 수정하는 값이 다음 경로에 있는 값 수정과 같은 역할을 한다 : D + space + 수정 + 맞춤 - 전체 축척 사용



5. 객체에 대한 정보 및 특성 확인 단축키 : **PR + space** / **CH + space**

* LTS + space 선축척 수정 기능의 경우, 도면 내에 있는 모든 선의 축척을 변경하는 기능이기 때문에, 이 기능이 필요하다.
* CH + space를 입력하여 나오는 선 수정 창에서, 원하는 선만 선택하여, 선의 종류, 스타일 및 축척 등을 수정하여 적용하면 된다.
* 치수선의 경우도 똑같이 적용하는 것이 가능하기 때문에, dimscale 등등의 파라미터를 수정할 수 있다.
  - 여기서, 공차 한계 상한 및 공차 한계 하한값 역시 지정이 가능하다.
  - 추가적으로, 공차 표시 방식 역시 모두 지정 가능하다.



6. 특성 복사 단축키 : **MA + space**

* 특성을 복사할 원본 객체를 먼저 선택하고, 대상 객체를 선택하면 같은 특성이 그대로 적용된다.
* 여러 치수선이 있는데, 먼저 하나의 치수선에 치수 머릿말을 기입하고, 다른 치수선에 MA 기능을 적용하게 되면, 똑같이 치수 머리말이 적용된다.
